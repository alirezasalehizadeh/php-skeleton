# PHP-Skeleton

My skeleton for PHP packages.

## Features


##  Requirements
PHP >= 8.2


## Getting Started
#### Installation:
via Composer:
```
composer create-project alirezasalehizadeh/php-skeleton project-name
```

## Usage

Keep a modern codebase with **Pint**:
```
composer lint
```

Run tests using **PEST**
```
composer test
```
